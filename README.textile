h1. Nakatype.js

Prototype.js extensions.

h2. Contents

h3. "Custom event for hash:changed":http://github.com/nakajima/nakatype/wikis/custom-event-for-hash-changed

Fires a custom Prototype event named @hash:changed@ when the window location hash fragment changes.

h3. "Better Edit in Place":http://github.com/nakajima/nakatype/wikis/better-edit-in-place-editable-js

A better edit in place script than the standard script.aculo.us solution.

h3. "Element controls":http://github.com/nakajima/nakatype/wikis/element-controls-element_controls-js

Create elements that control elements.

h3. "Classy Controls":http://github.com/nakajima/nakatype/wikis/classy-controls-classy-js

Simple class-based behaviors.

h3. "Callbacks":http://github.com/nakajima/nakatype/wikis/callbacks

Add before/after callbacks for any method for any object.
